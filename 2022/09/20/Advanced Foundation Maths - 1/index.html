Function: The relation between two sets

$f(x)=-\sqrt{x} \ \ \ \ D_f={1,4,6,16,25}$ function equals to minus square root of x

$f:A->B$ A is domain and B is co-domian

  • 对应关系
    • one->one 有且仅有一个x对应一个y
    • many->one 存在两个及以上x对应一个y
  • co-domain
    • range==co-domain: onto
    • range < co-domain: into
Composition of function

$(fog)(x)=f(g(x))$

Read: f over g x equals to f of g(x)

$R_f ⊆ D_g$

Inverse function

y is a subject to formula -> x is a subject to formula

Modulus function

$f(x)= |x|$ function of x equals to modulus

Exponential function

$y=f(x)=a^x$ a to the power of x

Logarithmic function

$y=f(x)=log_ax$ x is logarithym of y to base a

Translation
  • Move:
    • $f(x)\rightarrow f(x)+a$ f(x) shifted upward/downward by a distance of a units
    • $f(x)\rightarrow f(x+a)$ f(x) shifted to the right/left by a distance of a units
  • Stretch: $f(x) \rightarrow kf(x) \ \ k>1$ k·f(x) is stretching of the graph of f(x) vertically /vɜːtɪkli/ by a factor of k
  • Compressions: $f(x) \rightarrow kf(x) \ \ 0<k<1$ k·f(x) is compressioning of the graph of f(x) vertically by a factor of $1\over k$
Polynomial function 多项式

  • It can be expresseded as a sum of finitely many terms of the form $ax^n$

  • read: x raised to n or x to the pwer of n -> $x^n$

  • Degree = n

    | Degree | Name |
    | ——— | ————- |
    | 0 | Constant |
    | 1 | Linear |
    | 2 | Quadratic |
    | 3 | Cubic |
    | 4 | Quartic |
    | 5 | Quintic |

Example

  1. Prove that $(\frac{x^p}{x^q})^{p^2+pq+q^2}·(\frac{x^q}{x^r})^{q^2+qr+r^2}·(\frac{x^r}{x^p})^{r^2+rp+p^2} = 1$
